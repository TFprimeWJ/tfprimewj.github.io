Create folder & file

Create root folder and app.js

mkdir FirstExpressApp

cd FirstExpressApp

touch app.js

Install express

Install node package: express

npm install express

terminal output when installing express:

Now, there are 3 file/folder in the root folder:

app.js node_modules package-lock.json

Open node_modules:

cd node_modules, ls

Here we see:

Even though we only install express package, there are so many other folders generated along with the process.

Inside app.js

This code will start a server at local host port 3000.

We can access the server by typing:

http://localhost:3000

into our browser.

Create GET response

NOTE: Everytime we change app.js. be sure to restart the server. Press ctrl + c to close the server and type node app.js to start the server again.

Now, we can type 3 different url into the browser.

  • localhost:3000/
  • localhost:3000/bye
  • localhost:3000/dogs
