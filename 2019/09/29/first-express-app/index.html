Udemy Web Development: 265 Our First Express App

Create folder & file

Create root folder and app.js

mkdir FirstExpressApp

cd FirstExpressApp

touch app.js

Install express

Install node package: express

npm install express

Import express

Inside app.js

var express = require("express"); // Typo warning: require, not required
var app = express();

app.xxx // xxx stands for its functions

Start a server

Inside app.js

var express = require("express");
var app = express();

app.listen(3000, function(){
    console.log("Server has started!!!");
});

This code will start a server at local host port 3000.

We can access the server by typing:

http://localhost:3000

into our browser.

Create GET response

var express = require("express");
var app = express();

// "/" => "Hi there!"
app.get("/", function(req, res){
    res.send("Hi there!");   
});

// "/bye" => "Goodbye!"
app.get("/bye", function(req, res){
    res.send("Goodbye");
});

// "/dog" => "MEOW!"
app.get("/dogs", function(req, res){
    console.log("Someone has made a request to /dogs!!!");
    res.send("MEOW!");
});

// Tell Express to listen for requests (start server)
app.listen(3000, function(){
    console.log("Server has started!!!");
});

NOTE: Everytime we change app.js. be sure to restart the server. Press ctrl + c to close the server and type node app.js to start the server again.

Now, we can type 3 different url into the browser.

  • localhost:3000/
  • localhost:3000/bye
  • localhost:3000/dogs
