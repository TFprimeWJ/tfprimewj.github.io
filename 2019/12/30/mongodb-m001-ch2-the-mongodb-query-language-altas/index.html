MongoDB Shell

MongoDB Shell is a MongoDB text-based client.

2. Installing the MongoDB Shell (OSX/Linux)

Download link: https://www.mongodb.com/download-center/enterprise

在终端输入echo $PATH可以看到目前所有的路径，这些路径就是计算机寻找可执行程序的时候逐一访问的路径。

下面我们把MongoDB添加到这个路径中。

打开.zshrc，在最下面添加一行

export PATH="~/mongodb-macos-x86_64-enterprise-4.2.2/bin:$PATH"

冒号:：表示如果前面的路径没有我们想要的程序，那么就在冒号后面的路径继续搜索。而在上面的操作中我们也看到，$PATH正是现在所有的路径。所以这部操作其实就是把以前所有的路径添加到MongoDB路径的后面。

其实就是Concatenating。
