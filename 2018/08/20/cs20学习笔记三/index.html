3. Review

3.1 Computation Graph

TensorFlow separates definition of computations from their execution Phase 1: assemble a graph Phase 2: use a session to execute operations in the graph.

3.2 TensorBoard

3.3 tf.constant and tf.Variable

Constant values are stored in the graph definition Sessions allocate memory to store variable values

3.4 tf.placeholder and feed_dict

Feed values into placeholders with a dictionary (feed_dict) Easy to use but poor performance

3.5 Avoid lazy loading

  1. Separate the assembling of graph and executing ops
  2. Use Python attribute to ensure a function is only loaded the first time it’s called

4.2 What is the model about?
