Pandas学习-Series()转成str操作，读取gps

@(我的第一个笔记本)

Talk less, code more

在DataFrame中取一列转化成Series类型，对Series类型进行字符串分割，便可以得到坐标信息。

import pandas as pd

data_file = 'matchResult_10.txt'
names = 'app_name,app_kind,app_type,ip,nation,gps,size'
names = names.split(',')

data = pd.read_csv(data_file, header=None, names=names)

data['gps_xy'] = data['gps'].str.split('#')

print(data['gps_xy'])
